# Portfolio

Canonical index of organization projects and products.

## Rules

- one entry per repository or independently versioned product;
- reference the repository as the source of implementation truth;
- keep only stable purpose, ownership, lifecycle, and dependency information here;
- do not duplicate repository README files, roadmaps, issues, or transient status.

## Initial integration targets

Entries are added only through reviewed changes after the owning repository and its public contract are confirmed.

## Dependencies

- `identity`
- `governance`
